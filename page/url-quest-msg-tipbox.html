      <URL>, <QUEST>, <MSG>, <TIPBOX>

      The tags <URL>, <QUEST>, <MSG>, and <TIPBOX> are special HTML-like tags used within the mes command to display interactive elements in the client's dialogue window. They allow for linking to external websites, quests, internal messages, and tip boxes, respectively.

      ðŸŒ <URL> Tag

      The <URL> tag is used to create a clickable link to an external website directly within an NPC dialogue. When a player clicks on this link, their default web browser will open to the specified URL.

      • Purpose: To direct players to external resources such as forums, wikis, or event pages.
      • Syntax: 
        <URL>Display Name<INFO>http://www.example.com/</INFO></URL>
      • Parameters:
        • Display Name: The text that will be displayed as the clickable link in the dialogue.
        • http://www.example.com/: The full URL of the website to open.

      ðŸ“œ <QUEST> Tag

      The <QUEST> tag allows you to create a clickable link to a specific quest within the game's quest log. Clicking this link will open the player's quest window and navigate to the details of the specified quest ID.

      • Purpose: To guide players to relevant quests, provide quest hints, or confirm quest objectives.
      • Syntax: 
        <QUEST>Quest Display Name<INFO>Quest ID</INFO></QUEST>
      • Parameters:
        • Quest Display Name: The text displayed as the clickable quest link.
        • Quest ID: The numeric ID of the quest as defined in the server's quest database.

      ðŸ’¬ <MSG> Tag

      The <MSG> tag is used to display a message from the client's internal msgstringtable.txt file (or its equivalent in newer clients). This allows for displaying predefined client-side messages without hardcoding them directly into the script.

      • Purpose: To utilize client-side localized messages, often for system-generated or common dialogue phrases.
      • Syntax: 
        <MSG>Message ID</MSG>
      • Parameters:
        • Message ID: The numeric ID corresponding to an entry in the client's msgstringtable.txt.

      ðŸ’¡ <TIPBOX> Tag

      The <TIPBOX> tag displays a small, informational tip box on the client's screen. This is typically used for short, contextual hints or explanations.

      • Purpose: To provide quick tips or additional information to the player without interrupting the main dialogue flow.
      • Syntax: 
        <TIPBOX>Display Name<INFO>Tip ID</INFO></TIPBOX>
      • Parameters:
        • Display Name: The text that will be displayed as the clickable trigger for the tip box.
        • Tip ID: The numeric ID of the tip to display. The content of the tip is defined client-side.
      Example Code:
      //===== rAthena Script =======================================
//= Interactive_NPC
//===== By: rAthena AI Assistant ============================
//= Demonstrates the usage of URL, QUEST, MSG, and TIPBOX tags.
//============================================================
prontera,156,180,4	script	Interactive NPC	803,{
    mes "Welcome! I can show you some interactive features.";
    next;
    mes "Visit our official website: <URL>rAthena.org<INFO>http://rathena.org/</INFO></URL> for more info.";
    next;
    mes "Check out your first quest: <QUEST>Novice Quest<INFO>1001</INFO></QUEST>.";
    next;
    mes "Here's a common message: <MSG>1</MSG>.";
    next;
    mes "Need a hint? Click here: <TIPBOX>Basic Tip<INFO>1</INFO></TIPBOX>.";
    close;
}

      In this script:

      • The Interactive NPC is placed in Prontera and serves as a demonstration for these interactive tags.
      • The first mes command uses the <URL> tag to create a clickable link to "rAthena.org", which will open the specified URL in the player's web browser.
      • The second mes command uses the <QUEST> tag to create a clickable link named "Novice Quest" that, when clicked, will open the client's quest log to quest ID 1001.
      • The third mes command uses the <MSG> tag to display a message corresponding to message ID 1 from the client's msgstringtable.txt.
      • The fourth mes command uses the <TIPBOX> tag to create a clickable text "Basic Tip" that, when clicked, will display a tip box associated with tip ID 1.
      • This script effectively showcases how to embed various interactive elements directly within NPC dialogue lines using these specialized tags.
