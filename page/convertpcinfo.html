      convertpcinfo()

      ü§î What is convertpcinfo?

      The convertpcinfo function is a utility function in rAthena scripting that allows you to retrieve specific information about a character, given their character ID, account ID, or player name. It acts as a converter, transforming one type of player identifier into another piece of information.

      üéØ Purpose of convertpcinfo?

      The primary purpose of convertpcinfo is to obtain various details about a player, such as their name, character ID, or account ID, when you only have one of these identifiers. This is particularly useful when you need to cross-reference player data or perform actions that require a specific type of ID or name.

      üìù Syntax of convertpcinfo?

      The convertpcinfo function has the following syntax:

      • convertpcinfo(<char_id>,<type>)
      • convertpcinfo(<account_id>,<type>)
      • convertpcinfo(<player_name>,<type>)

      üìã Syntax Parameter Explanation:

      • <char_id>, <account_id>, or <player_name>: This is the identifier of the character you want to query. You can provide either the character's unique ID, their account ID, or their exact player name (as a string).
      • <type>: This parameter determines what specific information about the character will be returned. It accepts the following constants:
        • CPC_NAME: Returns the character's name (as a string).
        • CPC_CHAR: Returns the character's ID (as a number).
        • CPC_ACCOUNT: Returns the account ID associated with the character (as a number).

      If the character is not found (or not online) when requesting information, CPC_NAME will return an empty string, and other types will return 0.

      üí° How to Use convertpcinfo?

      To use convertpcinfo, you simply call the function with the character's identifier and the type of information you wish to retrieve. The returned value can then be stored in a variable or used directly in expressions.

      Example Code:
      //===== rAthena Script =======================================
//= PlayerInfoConverter
//===== By: rAthena AI Assistant ============================
//= Demonstrates the use of convertpcinfo to get player data.
//============================================================
splendide,194,145,5	script	PlayerInfoConverter	903,{
    mes "Hello, adventurer! I can convert player information for you.";
    mes "";
    mes "Please enter the name of a player you want to look up:";
    input .@target_name$;

    // Attempt to get the character ID from the provided name
    .@char_id = convertpcinfo(.@target_name$, CPC_CHAR);

    if (.@char_id == 0) {
        mes "Sorry, I couldn't find an online player named '" + .@target_name$ + "'.";
    } else {
        mes "Found player: " + .@target_name$ + ".";
        mes "";
        mes "--- Converted Information ---";
        mes "Character ID: " + .@char_id + ".";
        mes "Account ID: " + convertpcinfo(.@char_id, CPC_ACCOUNT) + ".";
        mes "Character Name (from ID): " + convertpcinfo(.@char_id, CPC_NAME) + ".";
        mes "";
        mes "--- Using Account ID to get info ---";
        .@account_id = convertpcinfo(.@char_id, CPC_ACCOUNT);
        mes "Character Name (from Account ID): " + convertpcinfo(.@account_id, CPC_NAME) + ".";
        mes "Character ID (from Account ID): " + convertpcinfo(.@account_id, CPC_CHAR) + ".";
    }

    close;
}

      In this script:

      • The script first prompts the player to input a target player's name using input .@target_name$;.
      • It then attempts to convert this player name into a character ID using convertpcinfo(.@target_name$, CPC_CHAR). If the player is not found or offline, this will return 0.
      • An if statement checks if a valid character ID was retrieved.
      • If the character is found, the script proceeds to display various pieces of information:
        • It directly shows the retrieved character ID.
        • It uses convertpcinfo(.@char_id, CPC_ACCOUNT) to get the account ID from the character ID.
        • It uses convertpcinfo(.@char_id, CPC_NAME) to re-confirm the character's name using their character ID.
      • Finally, it demonstrates retrieving the character name and character ID by starting with the account_id using convertpcinfo(.@account_id, CPC_NAME) and convertpcinfo(.@account_id, CPC_CHAR).
      • If the player is not found, a corresponding message is displayed.
