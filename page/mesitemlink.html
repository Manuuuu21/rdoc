      mesitemlink, <ITEM><INFO>

      What is mesitemlink?

      The mesitemlink command is used to generate a clickable item link string within NPC dialogue messages. This allows players to view item descriptions directly from the chat window.

      ‚ú® Purpose of mesitemlink

      The primary purpose of mesitemlink is to enhance player interaction within NPC dialogues by making item names clickable. When a player clicks on an item link generated by this command, it opens the item's description window in the game client, providing immediate information about the item without needing to search for it in the inventory or database.

      üìù Syntax of mesitemlink

      The syntax for the mesitemlink command is as follows:

      mesitemlink(<item_id>{,<use_brackets>{,<display_name>}}); mesitemlink("<item_name>"{,<use_brackets>{,<display_name>}});

      üîç Syntax Parameter Explanation

      • <item_id> or "<item_name>": This is the mandatory parameter that specifies the item to link. You can use either the item's database ID number or its English name (enclosed in double quotes).
      • <use_brackets>: This is an optional boolean parameter (true or false).
        • If set to true (default), the item link will be displayed surrounded by square brackets (e.g., [Red Potion]).
        • If set to false, the brackets will be omitted (e.g., Red Potion).
      • "<display_name>": This is an optional string parameter.
        • If provided, this custom name will be displayed as the clickable text for the item link instead of the item's default name from the database. This is useful for shortening long names or providing context.
      Example Code:
      //===== rAthena Script =======================================
//= ItemLink_NPC
//===== By: rAthena AI Assistant ============================
//= Demonstrates the usage of the mesitemlink command.
//============================================================
prontera,156,180,4	script	ItemLink NPC	867,{
    mes "Hello adventurer! I can show you some interesting items.";
    next;
    mes "Have you ever used a " + mesitemlink(501) + "?";
    mes "Or perhaps a " + mesitemlink("Blue Potion", false) + "?";
    mes "This is a custom link for a " + mesitemlink(1201, true, "Shiny Knife") + ".";
    next;
    mes "Click on the item names to see their descriptions!";
    close;
}

      In this script:

      • The NPC named ItemLink NPC is placed in Prontera at coordinates 150,150.
      • When a player interacts with the NPC, it displays a series of messages.
      • The first mes command demonstrates linking a Red Potion (item ID 501) using its default display name and with brackets (default behavior).
      • The second mes command links a Blue Potion (using its English name) but explicitly sets use_brackets to false, so it appears without brackets.
      • The third mes command links a Knife (item ID 1201) but provides a custom display name, "Shiny Knife", which will be shown as the clickable text.
      • This script effectively showcases how to present clickable item links in NPC dialogues, allowing players to easily access item information.

      What is <ITEM><INFO>? ü§î

      The <ITEM><INFO> tag is an HTML-like syntax used within mes commands in rAthena scripts to create clickable item links in NPC dialogue windows. When a player clicks on such a link, it typically opens a preview window for the specified item, displaying its description and stats.

      Purpose of <ITEM><INFO> üéØ

      The primary purpose of <ITEM><INFO> is to allow script developers to embed interactive item references directly into NPC conversations. This enhances the player experience by providing immediate access to item information without needing to manually search for the item or open an inventory. It's particularly useful for quest NPCs, shopkeepers, or informational NPCs that need to refer to specific items.

      Syntax of <ITEM><INFO> üìù

      The syntax for using <ITEM><INFO> is as follows: <ITEM>Display Name<INFO>Item ID</INFO></ITEM>

      Syntax Parameter Explanation üìñ

      • Display Name: This is the text that will be visible and clickable in the NPC dialogue window. It's what the player sees as the item link.
      • Item ID: This is the numerical ID of the item as defined in the server's item database (e.g., db/item_db.yml). This ID tells the client which item's information to display when the link is clicked.

      Note: In older client versions (prior to 2015), the tag name was <ITEMLINK> instead of <ITEM>. rAthena introduced the mesitemlink command to handle this compatibility automatically, which is the recommended way to create item links now.

      Example Code:
      //===== rAthena Script =======================================
//= ItemLink_NPC
//===== By: rAthena AI Assistant ============================
//= Demonstrates the usage of <ITEM><INFO> for item links.
//============================================================
prontera,156,180,4	script	ItemLink_NPC	767,{
    mes "Hello, adventurer! I can tell you about some useful items.";
    mes "For example, have you ever used a <ITEM>Red Potion<INFO>501</INFO></ITEM>?";
    mes "It's a basic healing item. Or perhaps a <ITEM>Butterfly Wing<INFO>602</INFO></ITEM> for quick travel?";
    mes "You can click on the item names to see their details.";
    close;
}

      In this script:

      • The script defines an NPC named ItemLink_NPC at splendide coordinates 194,145.
      • When a player interacts with this NPC, it displays a series of messages using the mes command.
      • Within these messages, the <ITEM>Display Name<INFO>Item ID</INFO></ITEM> syntax is used to create clickable links for "Red Potion" (Item ID 501) and "Butterfly Wing" (Item ID 602).
      • When a player clicks on "Red Potion" or "Butterfly Wing" in the NPC dialogue, their respective item information windows will pop up on the client.
