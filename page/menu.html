      menu

      The menu command in rAthena scripting is a fundamental tool for creating interactive dialogues with players. It allows you to present a list of selectable options, enabling players to make choices that influence the script's flow. Let's break down its usage:

      What is menu? ü§î

      The menu command displays a selectable menu to the invoking character. It's used to present choices to the player, and based on their selection, the script's execution can be directed to a specific label.

      Purpose of menu üéØ

      The primary purpose of menu is to:

      • Offer Choices: Present players with multiple options to choose from during an NPC interaction.
      • Control Script Flow: Direct the script's execution to different sections (labels) based on the player's selection.
      • Gather Input: Implicitly gather a numerical input (the selected option number) via the special @menu variable.

      Syntax of menu üìù

      The syntax for the menu command is as follows:

      menu "<option_text>","<target_label>"{,"<option_text>","<target_label>",...};

      Parameter Explanation:

      <option_text>: The text displayed for each menu option.

      <target_label>: The label within the script where execution will jump if this option is selected.

      If <target_label> is '-', the script continues immediately after the menu command if that option is chosen.

      Options can be grouped using a colon (:) within <option_text> (e.g., "Option A:Option B").

      An empty string "" as an option will hide that option from the display.

      How to use menu üí°

      When a player selects an option, the menu command automatically sets a special temporary character variable, @menu, to the number of the option chosen (starting from 1). You can then use this variable in conditional statements (if, switch) to control the script's behavior.

      Example Code:
      //===== rAthena Script =======================================
//= Menu_Example_NPC
//===== By: rAthena AI Assistant ============================
//= Demonstrates the usage of the 'menu' command.
//============================================================
payon,105,314,4	script	Menu_Example_NPC	667,{
    mes "Hello, adventurer! What would you like to do today?";
    next;
    menu "Check my level",L_CheckLevel,"Check my Zeny",L_CheckZeny,"Learn about rAthena",L_LearnAboutrAthena,"Exit",L_Exit;

L_CheckLevel:
    mes "Your current Base Level is: " + BaseLevel;
    next;
    mes "Is there anything else I can help you with?";
    next;
    goto L_MainMenu;

L_CheckZeny:
    mes "You currently have " + Zeny + " Zeny.";
    next;
    mes "Is there anything else I can help you with?";
    next;
    goto L_MainMenu;

L_LearnAboutrAthena:
    mes "rAthena is an open-source Ragnarok Online server emulator.";
    mes "It allows you to create and customize your own server.";
    next;
    mes "Is there anything else I can help you with?";
    next;
    goto L_MainMenu;

L_Exit:
    mes "Goodbye! Come back anytime.";
    close;

L_MainMenu:
    menu "Check my level",L_CheckLevel,"Check my Zeny",L_CheckZeny,"Learn about rAthena",L_LearnAboutrAthena,"Exit",L_Exit;

OnInit:
    // This label runs when the script is loaded.
    // No specific initialization needed for this simple menu NPC.
    end;
}

      In this script:

      • A main menu with four options.
      • Each option jumps to a specific label (L_CheckLevel, L_CheckZeny, L_LearnAboutrAthena, L_Exit).
      • After displaying information, the script returns to the main menu using goto L_MainMenu; to allow for further interaction.
      • The L_Exit label simply closes the NPC dialogue.
