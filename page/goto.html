      goto

      The goto command in rAthena scripting is a control flow statement that allows you to jump directly to a specified label within the current script.

      What is goto? üöÄ

      The goto command is a basic control flow statement that unconditionally transfers program control to a specified label. When goto is executed, the script immediately jumps to the line marked with the target label and continues execution from there.

      Purpose of goto üéØ

      The primary purpose of goto is to:

      • Alter Flow: Change the normal sequential execution flow of a script.
      • Jump to Sections: Directly move to a specific section of code identified by a label.

      While goto can be used to alter script flow, it is generally discouraged in modern programming practices as it can lead to complex and hard-to-read 'spaghetti code'. Structured control flow statements like if, else if, switch, for, and while are usually preferred for better readability and maintainability.

      Syntax of goto üìù

      The syntax for the goto command is straightforward:

      goto <label>;

      Parameter Explanation:

      <label>: The name of the label within the current script to which the execution should jump. Labels must be defined within the same script block.

      How to use goto üí°

      You define a label by placing its name followed by a colon (:) on a line by itself. When the goto command is encountered, the script searches for the specified label and resumes execution from that point. Any code between the goto command and the target label will be skipped.

      Example Code:
      //===== rAthena Script =======================================
//= Goto_Example_NPC
//===== By: rAthena AI Assistant ============================
//= Demonstrates the usage of the 'goto' command.
//============================================================
prontera,156,180,4	script	Goto_Example_NPC	668,{
    mes "Welcome! I will show you how 'goto' works.";
    next;

    mes "This is the first message.";
    next;

    // Using 'goto' to jump to a different section of the script
    goto SkipSection;

    mes "This message will be skipped because of 'goto'.";
    next;

SkipSection:
    mes "This is the message after the 'goto' jump.";
    next;

    // Example of conditional goto
    if (BaseLevel < 10) {
        mes "Your Base Level is low. Let's go to the novice advice section.";
        next;
        goto NoviceAdvice;
    } else {
        mes "Your Base Level is good. No novice advice needed.";
        next;
        goto EndScript;
    }

NoviceAdvice:
    mes "Remember to train hard and level up!";
    next;

EndScript:
    mes "Script execution finished.";
    close;

OnInit:
    // This label runs when the script is loaded.
    end;
}

      In this script:

      • An unconditional goto that skips a mes command.
      • A conditional goto within an if statement, directing the player to different sections based on their BaseLevel.
      • Labels (SkipSection, NoviceAdvice, EndScript) are used as jump targets for the goto command.
