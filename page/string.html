      String

      What is a String?

      In rAthena scripting, a string is a sequence of characters used to store text. It can contain letters, numbers, symbols, and spaces.

      Purpose of Strings?

      Strings serve various purposes in rAthena scripts, including:

      • Storing and displaying messages to players (e.g., NPC dialogues, announcements).
      • Storing player names, item names, map names, or other descriptive text.
      • Handling user input that involves text.
      • Constructing dynamic variable names or event labels.

      Syntax of Strings?

      In rAthena scripting, strings are enclosed within double-quotes ("").

      • Example: "Hello World"
      • Strings can be concatenated (joined together) using the + operator.
      Example Code:
      //===== rAthena Script =======================================
//= String_Example_NPC
//===== By: rAthena AI Assistant ============================
//= Demonstrates basic string usage and manipulation.
//============================================================
dewata,197,177,4	script	String_Example_NPC	727,{
    mes "Hello, adventurer!";
    mes "What is your name?";
    next;
    input .@player_name$;
    
    if (.@player_name$ == "") {
        mes "You didn't tell me your name.";
        close;
        return;
    }
    
    mes "Nice to meet you, " + .@player_name$ + "!";
    mes "Did you know your current map is " + strcharinfo(3) + "?";
    next;
    
    .@item_name$ = getitemname(501); // Get the name of Red Potion
    mes "I have some " + .@item_name$ + "s if you need them.";
    next;
    
    mes "This is a string with an escaped quote: ""Hello"".";
    next;
    
    // --- Substring Example ---
    .@full_string$ = "Ragnarok Online";
    .@sub_string$ = substr(.@full_string$, 0, 7); // Extracts "Ragnarok"
    mes "From '" + .@full_string$ + "', the substring from index 0 to 7 is: " + .@sub_string$ + ".";
    next;
    
    // --- Explode Example ---
    .@data_string$ = "Apple,Banana,Grape,Orange";
    explode(.@fruit_array$, .@data_string$, ",");
    
    mes "I split '" + .@data_string$ + "' by comma.";
    mes "The first fruit is: " + .@fruit_array$[0] + ".";
    mes "The third fruit is: " + .@fruit_array$[2] + ".";
    next;
    
    // --- Implode Example (for completeness, though not explicitly asked) ---
    .@combined_string$ = implode(.@fruit_array$, " - ");
    mes "And combining them with ' - ' gives: " + .@combined_string$ + ".";
    next;
    
    // --- strtoupper and strtolower Examples ---
    .@mixed_case_string$ = "rAthena Scripting";
    .@upper_string$ = strtoupper(.@mixed_case_string$);
    .@lower_string$ = strtolower(.@mixed_case_string$);
    mes "Original: " + .@mixed_case_string$ + ".";
    mes "Uppercase: " + .@upper_string$ + ".";
    mes "Lowercase: " + .@lower_string$ + ".";
    next;
    
    // --- getstrlen Example ---
    .@length = getstrlen(.@mixed_case_string$);
    mes "The length of '" + .@mixed_case_string$ + "' is " + .@length + " characters.";
    close;
}

      In this script:

      It serves as a demonstration of various string manipulation functions available in rAthena scripting.

      • Greets the player and asks for their name.
      • Validates input: Checks if the player entered a name.
      • Displays player's name and current map using strcharinfo.
      • Shows an item name using getitemname.
      • Demonstrates string escaping for double quotes.
      • Uses substr to extract a portion of a string.
      • Uses explode to split a string into an array based on a delimiter.
      • Uses implode to join an array of strings back into a single string.
      • Converts string case using strtoupper (to uppercase) and strtolower (to lowercase).
      • Determines string length using getstrlen.
